# Audi(프리미엄 오너) 1인칭 경험글 템플릿 & E-E-A-T 체크리스트

> 목적: AdSense "가치 없는 콘텐츠" 판정을 뚫는 **차별화 콘텐츠**를 양산하기 위한 표준 틀.
> 핵심 원리 — 법규 설명은 누구나 쓰지만, **실제 영수증·계기판·직접 촬영 사진**은 당신만 쓸 수 있다.
> 구글 E-E-A-T의 첫 글자 **E(Experience, 실제 경험)** 를 매 글에서 증명하는 것이 목표.

---

## 1. 글 1편 표준 구조 (섹션 순서)

1. **도입 — "나의 상황" 한 문단**
   - 차종/연식/주행거리/구매 시점을 첫 3줄에 명시. 예: "2023년식 Audi Q7 55 TFSI, 현재 28,400km, 6개월째 직접 운행 중."
   - 이 글에서 다룰 "실제로 겪은 문제/궁금증"을 1인칭으로 제시.

2. **핵심 결론 먼저 (요약 박스)**
   - 바쁜 독자를 위한 3~5줄 요약. (이미 에디터에 `요약 목록(summaryList)` 블록 있음 — 활용)

3. **본문 — 실제 데이터/경험 (글의 80%)**
   - 소제목(H2) 3~5개로 분할. 각 H2마다 **숫자·사진·표 중 최소 1개**.
   - "직접 측정/직접 지불/직접 겪음"을 드러내는 문장 필수.

4. **객관 정보 보강 (E-E-A-T의 나머지)**
   - 공식 출처(제조사 매뉴얼, 보증약관, 법령, 정비지침) 1~2개 인용 + 링크. → 신뢰성(Trust).
   - LawBox/TipBox 블록 활용.

5. **비교/대안**
   - "다른 선택지와 비교했을 때" (예: 사제 vs 정품, 일반 주유 vs 고급유). 표로.

6. **결론 + 다음 글 내부 링크**
   - 관련 글 2~3개 내부 링크(체류시간↑). 광고 문의/구독 유도는 과하지 않게.

---

## 2. E-E-A-T 체크리스트 (발행 전 점검)

발행 버튼 누르기 전, 각 글이 아래를 **최소 개수 이상** 충족하는지 확인:

- [ ] **Experience(경험)** — 직접 촬영 사진 **5장 이상** (계기판, 영수증, 부품, 차량 실물 등). 스톡 이미지 금지.
- [ ] **Experience** — 실제 숫자 **3개 이상** (실연비 km/L, 실제 지불 금액 원, 주행거리, 측정 일자).
- [ ] **Expertise(전문성)** — 표 또는 비교 데이터 **1개 이상**.
- [ ] **Authoritativeness(권위)** — 공식 출처(매뉴얼/법령/보증약관) 인용 **1개 이상** + 출처 링크.
- [ ] **Trust(신뢰)** — 단점·실패·후회도 솔직히 서술 (광고성 일색이면 감점).
- [ ] 본문 길이 **한글 4,000자 이상**.
- [ ] 제목/도입에 차종·연식·주행거리 등 **구체 정보** 포함.
- [ ] 메타 description 직접 작성 (자동 생성 의존 X).
- [ ] 썸네일 = 직접 촬영 사진.
- [ ] 내부 링크 2개 이상 / 관련 글 연결.

> 규칙: 위에서 **Experience 항목 2개(사진·숫자)는 필수**. 이게 빠지면 다른 블로그와 똑같아진다.

## 3. 바로 쓸 수 있는 Audi Q7 글 주제 10개

기존에 2편(실연비, CSP/ESP 보증) 있음. 아래로 8~10편까지 확장 권장:

1. **6개월 실유지비 총정리** — 주유·보험·소모품 영수증 전부 공개 (월별 표)
2. **Audi Q7 첫 정기점검 실비** — 정식 서비스센터 견적서 사진 + 항목별 설명
3. **고급유 vs 일반유 실연비 비교** — 같은 구간 2주씩 측정한 실데이터
4. **순정 타이어 마모 리포트** — 2만km 시점 트레드 측정 사진 + 교체 견적
5. **수입 대형 SUV 보험료 실제 청구액** — 가입 견적 비교 + 할증 경험
6. **Audi 커넥트/인포테인먼트 1년 사용기** — 실제 화면 캡처, 장단점
7. **겨울철 가솔린 대형 SUV 실연비 하락폭** — 여름 대비 측정 비교
8. **정품 부품 vs 애프터마켓** — 실제 교체하며 비교한 가격/품질
9. **Q7 적재공간 실측** — 줄자로 직접 잰 치수 + 캠핑/이사 활용기
10. **수입차 유지비, 사기 전 몰랐던 것 5가지** — 실패담 중심(Trust 강화)

> 교통법규 글과 섞어도 됩니다. "자동차·운전" 니치 안에서 일관되므로 AdSense 감점 요인 아님 — 오히려 1차 경험이 가점.

## 4. 기존 법규 글 업그레이드 미니 가이드

신규 글이 부담되면, 기존 18개 법규 글에 **오리지널 레이어**를 얹는 것도 효과적:

- "제가 실제로 ○○하다 겪은 일" 1인칭 일화 1문단 추가.
- 실제 과태료 고지서/단속 알림 문자/현장 사진(직접 촬영) 1~2장 추가.
- 단순 법령 나열 → "그래서 운전자는 실제로 어떻게 해야 하나" 실전 조언으로 재구성.
- 길이 4,000자+, 이미지 3장+ 로 보강.

## 5. 발행 후

- 글 5~8편 보강·추가 후, **최소 1~2주 뒤** AdSense 재신청.
- 신규 글은 GSC·네이버에 색인(수집) 요청.
- sitemap은 자동 갱신(`app/sitemap.ts`, 1시간 revalidate)되므로 별도 작업 불필요.
